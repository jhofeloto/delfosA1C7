Acerca del Proyecto Delfos A1 C7

Sistema inteligente para predicción de glucosa y clasificación de diabetes

Descripción del Proyecto

Delfos A1 C7 es un sistema avanzado que utiliza algoritmos de Machine Learning y Deep Learning para predecir los niveles de glucosa en ayunas y clasificar el estado de diabetes de los pacientes.

Objetivos
  • Predecir niveles de glucosa en ayunas (mg/dL)
  • Clasificar estado de diabetes: Normal, Prediabetes, Diabetes
  • Proporcionar herramientas de screening temprano
  • Apoyar la toma de decisiones clínicas
Dataset
  • Tamaño: 100 registros
  • Características: 43 variables clínicas
  • Variables objetivo: Glucosa (numérica) y Clase_DM (categórica)
  • División: 70% entrenamiento, 15% validación, 15% prueba

Metodología de Desarrollo

1. Preprocesamiento
  • Limpieza y manejo de valores faltantes
  • Codificación de variables categóricas
  • Normalización con StandardScaler
  • División estratificada del dataset
2. Entrenamiento
  • Múltiples algoritmos de ML/DL
  • Optimización de hiperparámetros
  • Validación con conjunto independiente
  • Evaluación exhaustiva con métricas estándar
3. Evaluación
  • Métricas de regresión: MSE, MAE, R²
  • Métricas de clasificación: Accuracy, Precision, Recall, F1
  • Matrices de confusión y análisis de errores
  • Comparación sistemática de modelos
4. Despliegue
  • Aplicación web con Flask
  • API RESTful para predicciones
  • Interfaz intuitiva y responsiva
  • Visualizaciones interactivas
Limitaciones Actuales
  • Dataset pequeño: 100 registros limitan la generalización
  • Desbalanceo de clases: Solo 4% de casos de diabetes
  • R² negativo: Regresión con rendimiento limitado
  • Validación limitada: Requiere validación clínica externa
Mejoras Futuras
  • Ampliar dataset: >1000 registros para mejor generalización
  • Feature engineering: Nuevas características derivadas
  • Validación cruzada: K-fold CV para robustez
  • Explicabilidad: SHAP values para interpretación
Descargo de Responsabilidad Médica

Este sistema es una herramienta de apoyo educativo y de investigación. Las predicciones generadas NO constituyen diagnóstico médico y no deben ser utilizadas como sustituto del criterio clínico profesional. Siempre consulte con un médico calificado para diagnósticos definitivos y decisiones de tratamiento.
